# Changelog

## 0.3.0

- Declared the first stable CLI and naming contract.
- Added `generate`, `check`, `lint`, and `version` commands while retaining 0.2 flag-only compatibility.
- Added `.image` units and image dependencies.
- Added lexical Quadlet drop-ins with reset semantics.
- Separated Quadlet filenames, OpenRC service names, and explicit Podman resource names.
- Added atomic writes and safe `--clean` handling.
- Added healthcheck, secret, device, privileged, and selected service lifecycle options.
- Added a complete hardened reference stack and migration documentation.
